# Manala v14 art evaluation

## Goal
Push the art much closer to a RuneScape Classic-inspired look without changing gameplay.

## What changed
- Replaced placeholder humanoids with more detailed character sprites using:
  - stronger outlines
  - face details
  - tunic/robe/armor silhouettes
  - weapon and shield readability
- Repainted object sprites with clearer forms and shading:
  - tree
  - rock
  - house
  - forge
  - anvil
  - bank booth
  - fish spot
- Added dedicated item icons:
  - bronze sword
  - raw fish
  - cooked fish
  - bronze ore
- Added terrain tiles:
  - grass
  - path
  - water

## Self-evaluation
This pass is **substantially closer** to the intended RSC-inspired direction than the earlier minimalist placeholders.
The silhouettes now read better at game distance, and the palette is much more in the classic beige / olive / dull-metal range.

## Is it sufficient?
**Close, but not final-perfect yet.**

What still separates it from a truly convincing RSC look:
1. Characters still need directional/facing variants for walking and interaction.
2. World objects need more sprite set variety (multiple tree/rock/house variants).
3. Ground needs richer tile mixing and edge transitions.
4. UI panels would benefit from matching iconography and portrait frames.
5. NPCs and players would benefit from tiny animation cycles.

## Recommendation
Use this as the new art baseline. Then do a v15 focused on:
- walk/idle/attack frame variants
- alternate tree/rock/house sprites
- stronger tile blending
- NPC portraits and dialogue art
