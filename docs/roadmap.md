# Manala Roadmap

## Vision
A RuneScape Classic-inspired online MMORPG that begins as a local browser prototype and evolves into a networked multiplayer experience.

## Current baseline
- v6b is the current stable baseline for combat and general interactions.

## v7 goals
- Timed fishing system
- Cook All / Smelt All batch loops
- Hybrid pseudo-isometric presentation
- First real sprite pipeline with RSC-inspired assets
- Better interaction labels and UI feedback

## v8+ goals
- Tile pathfinding
- Dialogue verbs like Talk / Attack / Mine / Chop
- Better bank/inventory drag-and-drop
- Quest system
- Server-authoritative multiplayer architecture
- Persistence layer
