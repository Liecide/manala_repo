# Manala v12 repo update

This update merges the stable gameplay loop back into the faux 3/4 renderer.

## Included
- Diagonal tile movement
- Camera pan (right-drag / arrow keys)
- Camera zoom (mouse wheel)
- Combat: simultaneous 3-hit exchange + 5 second immunity
- Timed fishing and mining
- Bank dialogue with vault grid
- Forge cook/smelt batch actions
- Anvil smithing dialogue
- Equipment UI, equip and unequip, equipped drops on death
- Save / load support

## Files to copy into your repo
- index.html
- src/game.js
- src/styles.css

## Notes
This is the first merged gameplay build in the faux RSC 3/4 view, so it focuses on keeping the working gameplay systems while replacing the renderer/camera.
