# Manala v15 repo update

Art-focused renderer upgrade for the public repo.

## Run

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## Included

- richer faux-RSC sprites
- pseudo-isometric renderer
- depth sorting
- hover verbs
- stepped click-to-move
- camera pan and zoom

This update is meant as the visual baseline for future integration with the fuller gameplay systems.
